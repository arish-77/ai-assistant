# AI Personal Assistant Web App

An AI-powered web application that enables users to interact with an intelligent assistant for chat, text processing, and content summarization. The application provides a clean and responsive interface while leveraging Large Language Models (LLMs) for real-time responses.

## 🚀 Features

- AI-powered conversational chatbot
- Text summarization
- Real-time response generation
- Responsive user interface
- RESTful API architecture
- Secure environment variable management
- Cloud deployment support

## 🛠️ Tech Stack

### Frontend
- HTML
- CSS
- JavaScript

### Backend
- Flask
- Python

### AI Integration
- Groq API
- Large Language Models (LLMs)

### Deployment
- Render
